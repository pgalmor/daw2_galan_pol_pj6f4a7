Description & Documentation

This is the internal website of Galan Hardware Store, where employees can create new orders, check for specific orders and view all the orders existing.

In the menu you'll see three options: New Order, View Order and View All Orders.
In New Order you'll see a form to add a new order, adding all the information of the customer and the items and quantities of the purchase. Once the order is sent you'll get the price of the order with the IVA added.
In View Order you'll have an text field where you can input a code and it will show you the order associated with that code. If there's no order with that code or there are no orders it will tell you so.
In View All Orders you'll see a list of all existing orders.
